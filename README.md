# phpfilemerger

Merge multiple PHP files into one in PHP, I have used this script for Codingame.

## Structure

```
bin/        
tests/
vendor/
```

```
src/        => put your files in this directory.        
public/     => result file will be find here.
```



## Install

Via Composer

``` bash
$ composer require smartinus44/phpfilemerger
```

## Usage

``` bash
$ composer start
```
